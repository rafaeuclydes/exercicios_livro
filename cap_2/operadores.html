Principais funções matemáticas da classe Math


Math.abs(num) Retorna o valor absoluto de um número.
Ex: Math.abs(-3) => 3.
Math.ceil(num) Arredonda o valor para cima.
Ex: Math.ceil(3.3) => 4.
Math.floor(num) Arredonda o valor para baixo.
Ex: Math.floor(8.7) => 8.
Math.pow(base, exp) Retorna a base elevada ao expoente.
Ex: Math.pow(2, 3) => 8.
Math.random() Retorna um número aleatório ente 0 e 1 com várias casa decimais.
Ex: Math.random() => 0.59941848761.
Math.round(num) Arredonda o valor para o número inteiro mais próximo.
Ex: Math.round(1.6) => 2.
math.sqrt(num) Retorna a raiz quadrada do número.
Ex: Math.sqrt(9) => 3.
